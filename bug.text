# 3、查询数据库报错：java.sql.SQLNonTransientConnectionException: Public Key Retrieval is not allowed
因为我的MySQL不是安装在本机的而且我使用的MySQl版本是8.0.28，所以我需要在连接数据库的字符串中指定这样一行：
allowPublicKeyRetrieval=true 允许客户端从服务器获取公钥
